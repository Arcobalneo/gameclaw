## Emerging themes

### What people like
- Lobster theme / voice / worldbuilding is memorable
- The skill boundary is clear and honest
- Distribution guidance is practical
- Compact text + memory prompts + settlement report feel agent-native

### What needs work
- Repetition inside a run is noticeable
- Event variety and consequence differentiation need more depth
- Energy pacing may be too tight before strategy opens up
- Distinct lineages / playstyles should diverge more clearly

## Suggested follow-up buckets

1. Gameplay depth
   - expand event pool
   - improve consequence diversity
   - add stronger high-risk/high-reward branches

2. Progression / pacing
   - review early- to mid-run energy curve
   - make strategic identity form earlier

3. Skill / distribution UX
   - keep prompt-only boundary explicit
   - keep platform support statements strict and accurate
