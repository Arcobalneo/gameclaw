# 2026-03-12 GDC 参考文档清洗计划

## Goal

将 `docs/references/gdc-game-design-insights.md` 清洗为一份可长期离线保存、可稳定复用的内部参考文档：
- 不包含任何外部 URL
- 不依赖外链跳转理解内容
- 只保留来源真实、可核实的事实性信息与明确标注为内部总结的启发
- 删除无法确认出处、带明显推断色彩或过度细节化的段落

## Scope

- 清洗 `docs/references/gdc-game-design-insights.md`
- 必要时同步更新其对应 plan 文档中的完成项描述
- 不扩展到整个 `docs/` 目录的其他计划文档

## Assumptions

- 当前联网检索工具不可用或受限时，以保守删减优先，而不是保留存疑内容
- 允许保留高置信度的基础事实（演讲名、讲者、年份、方法论主旨）
- 凡是无法在当前环境下重新核验的细节，一律降级为更保守表述或直接删除

## Risks

- 过度保守可能导致文档信息量下降
- 某些经典二手转述虽然大概率正确，但若无法快速复核，也需要删除
- 清洗后文风会从“教材式展开”变成“参考手册式摘要”
