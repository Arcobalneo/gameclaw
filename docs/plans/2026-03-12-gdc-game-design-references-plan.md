# 2026-03-12 GDC 游戏设计参考文档计划

## Goal

从 GDC（Game Developers Conference）相关资料中搜集 5 篇具有代表性的游戏设计思路分享，整理成结构化参考文档，放入 `docs/references/` 目录，供后续设计新 GameClaw 游戏时参考。

## Scope

- 搜索并筛选 5 篇高质量 GDC 演讲 / 文章
- 每篇提炼：标题 / 作者 / 年份 / 核心观点 / 对 GameClaw 的启发
- 整理为一个汇总 Markdown 文件：`docs/references/gdc-game-design-insights.md`
- 优先选择和以下主题相关的内容：
  - Roguelike / roguelite 设计
  - 程序生成（Procedural Generation）
  - 游戏手感与反馈（Game Feel / Juice）
  - 玩家决策与代入感
  - 系统平衡与进程设计

## Assumptions

- 不需要授权登录即可访问的 GDC Vault 免费内容或公开摘要
- 以文字整理摘要为主，不下载视频
- 5 篇内容覆盖面尽量广，不全是同一类型

## Risks

- GDC Vault 部分内容需要付费，应优先选免费公开的
- 摘要准确性依赖资料来源，应标注原始链接
