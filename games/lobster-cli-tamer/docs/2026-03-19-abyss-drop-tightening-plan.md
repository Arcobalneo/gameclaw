# 2026-03-19 深渊材料掉落分层收紧方案（plan）

## 背景

当前深渊风险循环已经开始收紧：

- 所有层都会累积深渊污染
- 污染会转化为疫病
- 退出深渊不再是低层零代价试错

但**材料掉落结构**仍然偏松，导致 loop 还不够纯：

- 普通层已经会掉 `甲核`
- 普通层也可能掉 `灵晶`
- `floor_drops` fallback 里还存在 `甲核 / 灵晶 / 潮石 / 灵光球`
- 这意味着玩家即使只刷低层，也仍然能持续拿到一部分后期养成材料

这和目标 loop 之间还存在偏差：

1. 大世界抓宠 / 练级 / 组队
2. 进入深渊承担风险
3. 在**更高层**深渊刷洗词条 / 升词条道具
4. 回工坊养成
5. 再挑战更高层

如果低层就能相对稳定地给出核心养成材料，玩家就会倾向于：

- 用更低风险的方法拖时间 farm
- 而不是为了更高收益去承担更高深渊风险

---

## 设计目标

### 核心目标

把深渊掉落体系收紧成：

- **低层可补给，不可毕业**
- **中层开始稳定产出核心养成材料**
- **高层 / Boss 才是高价值养成资源的主要来源**

### 期望结果

让玩家感知到以下事实：

- 想补捕捉资源、维持探索续航 → 浅层可以做
- 想真正洗词条、升词条、做高质量养成 → 必须往更深层走
- 想降低深渊损失 → 需要在大世界先抓更好的宠、练更好的队

也就是：

**大世界负责补队与练度，深渊高层负责核心养成资源。**

---

## 当前问题拆解

### 1. 普通层掉落过于“全能”

当前 `tower_config.json` 中：

- `normal` 会掉：
  - `甲核`
  - `灵晶`
  - `甲网`

问题：

- `甲核` 是重投词条直接材料
- `灵晶` 是升阶词条直接材料
- 这两者都属于工坊核心养成资源
- 如果普通层能稳定出，就会削弱“冲更深”这件事的必要性

### 2. `floor_drops` fallback 仍在泄露核心材料

当前 fallback 掉落池包含：

- `甲核`
- `灵晶`
- `潮石`
- `灵光球`

问题：

- 即使分层掉落没命中，也还有后备概率拿到核心材料
- 这会稀释分层设计本身

### 3. Boss / Elite 与普通层的收益差距还不够绝对

虽然当前数值上高层更赚，但结构上仍然是：

- 普通层：也能拿到养成资源
- Elite / Boss：只是拿更多

这会把 loop 做成“效率差异”，而不是“内容门槛差异”。

---

## 推荐方案：三段式掉落分层

我建议把深渊掉落划成 3 层：

### A. 浅层（普通层）= 生存补给层

用途：

- 给玩家一些持续探索的基础资源
- 允许浅层试探与续航
- 但不允许在这里长期完成后期养成

建议普通层只保留：

- `甲网`
- 少量 `净疫盐`（可选）
- 极少量通用低价值材料（如后续确实需要）

建议从普通层移除：

- `甲核`
- `灵晶`
- `潮石`
- `深渊灵晶`

一句话：

**普通层只能保命 / 续航，不能高效养成。**

---

### B. 中层（精英层）= 核心养成起点

用途：

- 让玩家第一次真正接触工坊后期资源
- 同时开始承受明显风险

建议精英层成为：

- `甲核` 的主稳定来源
- `灵晶` 的首个稳定来源
- `净疫盐` 的主要安全阀来源
- `精甲网` 的自然升级来源

建议精英层掉落结构：

- `甲核`：稳定掉，数量中等
- `灵晶`：稳定掉，数量偏少
- `净疫盐`：稳定小概率 / 小数量
- `甲网 / 精甲网`：继续保留
- 暂不掉 `潮石`
- 暂不掉 `深渊灵晶` 或仅极小概率

一句话：

**精英层开始真正喂养工坊，但还不是最终毕业区。**

---

### C. 高层（Boss / 深层）= 高级养成层

用途：

- 给真正承担高风险的玩家更好的词条成长效率
- 成为高阶工坊循环的主要资源入口

建议 Boss / 高层掉落结构：

- `甲核`：大量
- `灵晶`：大量
- `潮石`：从这里开始稳定掉
- `深渊灵晶`：从这里开始稳定掉
- `净疫盐`：保底或接近保底
- `灵甲网`：保留
- `灵光球`：保留在极低概率稀有奖励位

一句话：

**Boss / 高层才是“刷词条毕业材料”的主要地点。**

---

## 建议的具体收紧方向

### 方案表（建议版）

#### 普通层

保留：

- `甲网`
- （可选）`净疫盐` 低概率

移除：

- `甲核`
- `灵晶`
- `潮石`
- `深渊灵晶`
- `灵光球`

#### 精英层

保留并强化：

- `甲核`
- `灵晶`
- `净疫盐`
- `甲网`
- `精甲网`

限制：

- `潮石` 不在精英层稳定掉
- `深渊灵晶` 只做弱概率预告，不做稳定来源

#### Boss 层

稳定产出：

- `甲核`
- `灵晶`
- `潮石`
- `深渊灵晶`
- `净疫盐`
- `精甲网`

稀有产出：

- `灵甲网`
- `灵光球`

## 对现有配置的直接建议

### 1. 删掉 `floor_drops` 里的核心养成材料

建议处理：

- 从 `floor_drops` 移除：
  - `甲核`
  - `灵晶`
  - `潮石`
- 如果还需要 fallback，可以只保留：
  - `甲网`
  - `净疫盐`
  - `灵光球`（极低概率）

原因：

- fallback 不应绕过分层设计
- fallback 应该是“补充奖励”，不是“绕过门槛”

### 2. 普通层改成补给池，而非养成池

建议把当前 `normal`：

- `甲核: [1,3]`
- `灵晶: [0,1]`
- `甲网: [0,1]`

调整为类似：

- `甲网: [1,2]`
- `净疫盐: [0,1]`（可选）

即：

- 有补给
- 有续航
- 但没有后期工坊毕业资源

### 3. 把 `潮石` 明确抬升到 Boss 档

`潮石` 对应封印/解封，属于较高价值的定向养成资源。

建议：

- 不在普通层掉
- 不在精英层稳定掉
- 由 Boss / 更高层负责供应

这样玩家想做更精细的词条管理，就必须往更危险处走。

---

## 为什么这样更符合 game loop

### 1. 大世界重新变得有意义

如果浅层深渊不能稳定掉后期养成材料，那么玩家会更自然地：

- 回大世界抓更好的宠
- 补队伍
- 练级
- 调阵容

因为这才是“为更深深渊做准备”的正确路径。

### 2. 深渊高层重新成为真正目标

玩家会明确知道：

- 浅层只是试探 / 续航 / 过渡
- 真正要洗词条、升词条、做高级管理，必须去更危险的层

### 3. 永久死亡的存在会变得更有重量

如果高价值资源只在高风险层稳定产出，那么“减员 / 灭队”的代价就真正嵌入 loop：

- 不是随便赔一个低层 farm 时间
- 而是会真实打断高价值养成进程

这才和 permadeath 的设计力度匹配。

---

## 不建议的错误改法

### 错法 1：只是把普通层数值调低一点

问题：

- 结构没变
- 玩家仍然可以刷，只是效率慢一点
- 最终还是时间换资源

### 错法 2：把普通层掉率调很低，但不删来源

问题：

- 依然会给玩家“总能刷出来”的心理预期
- 不能形成真正的内容门槛

### 错法 3：所有资源都只给 Boss

问题：

- 太硬
- 精英层失去中段推进价值
- 过渡层断档，节奏会变得生硬

## 推荐实施顺序

### Phase 1：掉落结构收口（优先）

Checklist：

- [x] 清理 `floor_drops` 中绕过分层的核心养成材料
- [x] 普通层移除 `甲核 / 灵晶 / 潮石`
- [x] 精英层设为 `甲核 / 灵晶 / 净疫盐 / 精甲网` 的主入口
- [x] Boss 层设为 `潮石 / 深渊灵晶 / 高级网` 的核心入口

### Phase 2：深层收益强化（随后）

Checklist：

- [ ] 增加 Boss / 深层的数量优势
- [ ] 让高层单位时间收益显著优于浅层
- [ ] 让玩家明确感受到“越深越值”

### Phase 3：验证

Checklist：

- [ ] 脚本模拟浅层 farm 的单位时间收益
- [ ] 脚本模拟精英 / Boss 的单位时间收益
- [ ] 比较不同层数的工坊成长速度
- [ ] 用真实 agent 玩一轮，看是否还会优先选择低层 farm

---

## 最终建议

**我建议把“深渊材料掉落分层”真正做成内容门槛，而不只是效率梯度。**

最终应当让玩家形成下面这套稳定认知：

- 浅层：补给、续航、试探
- 精英层：核心养成起点
- Boss / 高层：高级养成主产地

这样 loop 才会闭得更紧：

**抓宠 / 练级 → 深渊承担风险 → 拿材料养成 → 冲更高层 → 失败后回大世界重建**

而不是退化成：

**低层慢慢刷也能把后期资源磨出来。**
