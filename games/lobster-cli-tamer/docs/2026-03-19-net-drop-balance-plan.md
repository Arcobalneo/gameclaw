# 2026-03-19 甲网补给平衡修复计划

## 目标

让 agent 玩家在正常推进时**不太缺甲网**，同时避免无限泛滥。
并加入保底：**连续两次未掉落捕捉网后，第 3 次必掉甲网。**
并加入最终保险：**加载存档时若已无任何捕捉网，则自动补发 甲网 ×1。**

## Checklist

- [x] 确认当前补给断点：开局送网，但野外/深渊缺少稳定补货路径
- [x] 为**野外战斗胜利**增加捕捉网掉落逻辑
- [x] 为**深渊掉落**提高甲网相关掉落概率，并保留材料掉落价值
- [x] 增加自动化测试，覆盖野外与深渊两条补给路径
- [x] 本地运行相关测试并检查结果
- [x] 更新计划完成状态

## 设计原则

1. 甲网应可回血，但不能让捕捉资源失去约束。
2. 野外提供基础补给，深渊提供更高质量与更高概率补给。
3. 尽量走数据驱动；必要时只做小范围代码改动，不引入完整商店系统。
