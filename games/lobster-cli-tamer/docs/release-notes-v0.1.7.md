# lobster-cli-tamer v0.1.7 — 深渊掉落分层收紧

本次版本继续强化核心 loop：

- 大世界抓宠 / 练级 / 组队
- 深渊承担风险
- 通过更深层深渊拿核心养成材料
- 回工坊洗词条 / 升词条
- 再挑战更高层

## 本次调整

### 1. 普通层改为补给层

普通层现在只承担续航与试探功能，主要掉落：

- 甲网
- 低概率净疫盐

普通层不再稳定掉落以下核心工坊材料：

- 甲核
- 灵晶
- 潮石

### 2. 精英层改为核心养成起点

精英层现在是核心工坊资源的第一稳定来源，主要掉落：

- 甲核
- 灵晶
- 净疫盐
- 甲网 / 精甲网

深渊灵晶只保留为较弱预告位，不再过早泛滥。

### 3. Boss 层继续作为高级养成主产地

Boss 层仍然是高级资源主来源，主要掉落：

- 甲核
- 灵晶
- 深渊灵晶
- 潮石
- 净疫盐
- 精甲网 / 灵甲网

### 4. fallback 掉落池收口

`tower_config.json` 的 `floor_drops` fallback 不再泄露核心工坊材料。

现在只保留：

- 灵光球（极低概率）

这样就不会通过 fallback 绕过分层设计。

## 结果

深渊掉落结构从“高层更赚”进一步收紧为：

- 浅层：可补给，但不能毕业
- 精英层：核心养成起点
- Boss / 高层：高级养成主产地

这让 permadeath、污染/疫病、工坊资源和深层挑战之间的关系更闭合。

## 验证

已通过本地测试，并使用真实 Linux 二进制包做分层掉落验证。
