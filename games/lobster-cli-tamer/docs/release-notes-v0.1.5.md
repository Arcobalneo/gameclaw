# lobster-cli-tamer v0.1.5 — 捕捉网补给保底

本次版本聚焦一个核心目标：**不让 agent / 玩家因为连续脸黑而被“没甲网”卡死主循环。**

## 更新内容

- 野外战斗胜利后，新增捕捉网掉落逻辑
  - 甲网：30% 概率掉落 1 个
  - 精甲网：6% 概率掉落 1 个
- 深渊掉落池加入甲网系补给
  - 普通层可掉甲网
  - 精英层可掉甲网 / 精甲网
  - Boss 层可掉甲网 / 精甲网 / 灵甲网
- 新增**共享保底机制**
  - 野外与深渊共用同一条保底计数
  - 连续 2 次未掉落任何捕捉网后，第 3 次**必掉甲网 ×1**
- 新增**最终保险机制**
  - 加载存档时，如果背包中已无任何捕捉网，则自动补发 **甲网 ×1**

## 设计结果

- 降低 agent 长局因断网导致的流程中断概率
- 保留资源管理与捕捉决策压力
- 避免出现“完全无网 → 主循环卡住”的死锁体验
